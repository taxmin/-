# 项目部署与使用指南

## 📦 快速开始（新用户）

### 方法1：复制 Python 环境（最快）

1. **从原项目复制 Python 环境**
   ```
   复制整个 Python38Portable 目录到新项目根目录
   ```

2. **运行程序**
   ```batch

   ```

### 方法2：自动安装环境（推荐）

1. **确保有网络连接**

2. **运行安装脚本**
   ```batch

   ```
   
   脚本会自动：
   - ✅ 检测 Python 环境
   - ✅ 安装所有依赖包
   - ✅ 修复 dxpyd 模块问题
   - ✅ 检查 OCR 环境

3. **运行程序**
   ```batch
   双击 run.bat
   ```

## 💡 最佳实践

### 对于开发者

1. **不要直接调用 Windows API**
   ```python
   # ❌ 错误：直接调用
   self.dx.KM.PressKey('esc')
   
   # ✅ 正确：使用封装函数
   安全按键('esc')
   ```

2. **所有新函数都要有异常保护**
   ```python
   def 新功能():
       try:
           # 业务逻辑
           pass
       except OSError as e:
           print(f"⚠️ 操作失败: {e}")
           return False
       except Exception as e:
           print(f"⚠️ 异常: {e}")
           return False
   ```

3. **使用 KM 锁保护键鼠操作**
   ```python
   from app.vmware_window_monitor import get_km_lock
   km_lock = get_km_lock(self.row)
   
   with km_lock:
       self.dx.KM.MoveTo(x, y)
       self.dx.KM.LeftClick()
   ```

### 对于用户

1. **首次使用**
   - 优先复制 `Python38Portable` 目录
   - 或者运行 `install_env.bat`

2. **遇到问题**
   - 先查看日志文件：`资源/日志/`
   - 运行 `install_env.bat` 修复环境
   - 查看详细文档：`dxpyd导入问题修复说明.md`

3. **保持更新**
   - 定期 `git pull` 获取最新代码
   - 每次更新后运行 `install_env.bat`
