# 项目开源保护策略

## 🎯 目标

**公开项目框架，保护核心逻辑**

让别人能够：
- ✅ 看到项目结构和代码风格
- ✅ 学习技术实现思路
- ✅ 运行基础功能演示

但无法：
- ❌ 获取你的核心算法和业务逻辑
- ❌ 直接复制用于商业用途
- ❌ 了解你的优化策略和技巧

---

## 🛡️ 三层保护体系

### 第一层：配置文件分离（已实施）

#### 敏感数据本地化

以下文件**不会**上传到 Git：

1. **config_local.py** - 本地配置
   ```python
   # VNC 端口映射（关键！）
   VNC_PORT_MAPPING = {
       0: ("127.0.0.1", 5900),
       1: ("127.0.0.1", 5901),
   }
   
   # 账号信息（绝对不能公开）
   ACCOUNT_CONFIG = {
       "5601": [("账号", "密码", "区服", "角色")],
   }
   ```

2. **core_strategy_local.py** - 核心策略
   ```python
   class CoreStrategy:
       def optimize_task_sequence(self, task_list):
           # 你的核心算法
           pass
   ```

3. **其他 *_local.py 文件** - 任何本地模块

#### 使用方法

```bash
# 1. 复制模板
cp config_local.example.py config_local.py

# 2. 编辑配置
vim config_local.py

# 3. 运行程序
python main.py
```

## 🚀 部署流程

### 开发者（你）

1. **开发时**
   ```bash
   # 正常工作流程
   git add app/ dxGame/ task_list/task.py
   git commit -m "更新任务框架"
   git push
   ```

2. **保护核心**
   ```bash
   # 确保本地文件不被提交
   git status
   # 确认没有 config_local.py, core_strategy_local.py 等
   ```

3. **发布前检查**
   ```bash
   # 检查是否有敏感信息
   grep -r "password" --include="*.py" .
   grep -r "账号" --include="*.py" .
   grep -r "VNC.*590" --include="*.py" .
   ```

### 用户（其他人）

1. **克隆项目**
   ```bash
   git clone <your-repo>
   cd project
   ```

2. **配置环境**
   ```bash
   # 复制配置模板
   cp config_local.example.py config_local.py
   
   # 编辑配置
   vim config_local.py
   # 填写 VNC 端口、账号信息等
   ```

3. **运行程序**
   ```bash
   # 安装依赖
   install_env.bat
   
   # 启动
   run.bat
   ```

## ⚠️ 注意事项

### 1. 不要硬编码敏感信息

```python
# ❌ 错误
VNC_PORT = 5900
ACCOUNT = "18408241173"
PASSWORD = "12300123."

# ✅ 正确
from config_local import VNC_PORT_MAPPING, ACCOUNT_CONFIG
```

### 2. 定期检查 Git 历史

```bash
# 检查是否有敏感文件曾被提交
git log --all --full-history -- "*password*"
git log --all --full-history -- "*config_local*"

# 如果发现了，需要从历史中彻底删除
git filter-branch --force --index-filter \
  'git rm --cached --ignore-unmatch config_local.py' \
  --prune-empty --tag-name-filter cat -- --all
```

### 3. 使用环境变量（可选）

```python
import os

# 从环境变量读取（更安全）
VNC_PASSWORD = os.environ.get('VNC_PASSWORD', '')
API_KEY = os.environ.get('GAME_API_KEY', '')
```

## 🎯 总结

**核心理念：**
- 公开**框架**，保护**内容**
- 公开**接口**，保护**实现**
- 公开**思路**，保护**细节**

**实施要点：**
1. ✅ 使用 `*_local.py` 模式隔离核心代码
2. ✅ `.gitignore` 正确配置
3. ✅ 提供配置模板 (`*.example.py`)
4. ✅ 清晰的文档说明
5. ✅ 定期审查代码

**最终效果：**
- 别人能看到你的技术水平
- 学到项目架构设计
- 但无法直接复制核心业务逻辑
- 需要自己实现关键算法
